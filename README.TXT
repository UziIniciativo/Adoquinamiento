Para compilar utilizar el comando
javac Adoquinador.java
Para correr utilizar el comando
java Adoquinador <k>
donde k es el tamaño de la región a adoquinar (nxn=2^k) y 1<k<10

La impresión de la matriz en la terminar se hace de manera automática 
al correr el programa si el valor de k es válido.
